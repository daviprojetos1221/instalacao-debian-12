# Configurando a BIOS/UEFI para Boot pelo Pendrive

1. Insira o pendrive no computador onde será instalado o Debian.
2. Reinicie o computador.
3. Pressione a tecla específica para acessar a BIOS/UEFI (geralmente `F2`, `F12`, `DEL`, `ESC` ou conforme indicado na tela inicial).
4. No menu da BIOS/UEFI, procure pela opção “Boot” ou “Ordem de boot”.
5. Coloque o pendrive como a primeira opção de boot.
6. Salve as alterações e saia (`F10` geralmente salva e sai).
7. O computador irá reiniciar e iniciar a partir do pendrive, mostrando o menu de instalação do Debian.
